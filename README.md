# Sistema de Clínica

## Status do projeto
Em desenvolvimento

## Tecnologias utilizadas
- Java
- NetBeans
- MySQL
- JDBC
- GitHub

## Time de desenvolvedores
- Marcelle Alves dos Santos

## Objetivo do software
O objetivo do sistema é auxiliar no gerenciamento de uma clínica, permitindo o cadastro e controle de pacientes e consultas.

## Funcionalidades do sistema
- Cadastro de pacientes
- Cadastro de consultas
- Listagem de pacientes
- Agendamento de consultas
- Navegação entre telas
